Pascal Compiler

(Spring 2019)

Built for a compiler construction class. The compiler is incomplete, however demonstrates many core components such as a working tokenizer, parser, symbol table, and runtime.

Technology

  • C
  • GCC
  • Pascal

Stages of the Compiler

Tokenizer

The compiler reads a text file and breaks down the text into tokens to iterate though. This includes keywords, numbers, symbols, and anyhting else the compiler might use.

Parser

The compiler then reads each token into a grammer, which ends up reaching leaf nodes that can be interpreted as instructions. These are written as opcodes in an output file

Runtime

The runtime is a stack machine, which has a code segment, data segment, and a stack. Computation is performed by executing instructions which modify the state of the stack.
